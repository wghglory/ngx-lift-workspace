The isEmpty utility function checks if a string, object, or array is empty. It returns true for empty values and false for non-empty values.

API Reference

isEmpty

Checks if a string, object, or array is empty.

Signature

Parameters

  • value: unknown

    The value to check. Can be a string, object, array, or other value.

Returns

true if the value is empty (empty string, empty object, empty array, null, or undefined), false otherwise.
