Overview

The ArrayJoinPipe is an Angular pipe that joins array elements into a string using a specified separator. It provides a convenient way to display array data as comma-separated or custom-delimited strings in templates.

API Reference

ArrayJoinPipe

Angular pipe that joins array elements into a string using a specified separator.

Transform Method

Signature:

transform(value: unknown, separator?: string): string | unknown

Parameters:

  • value: The array to join. If not an array, returns the value as-is.
  • separator: (Optional) The separator string to use between array elements. Defaults to ','.

Returns: A string containing the joined array elements, or the original value if not an array

Basic Usage

Use the pipe in your templates to join array elements. The default separator is a comma, but you can specify any custom separator.
