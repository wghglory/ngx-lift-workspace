Overview

The IsHttpsPipe is an Angular pipe that checks if a URL string uses the HTTPS protocol. It's useful for conditional rendering, validation, or security checks in templates.

A corresponding isHttps utility function is also available for use in TypeScript. See the URL utilities documentation for more details.

API Reference

IsHttpsPipe

Angular pipe that checks if a URL string uses the HTTPS protocol.

Transform Method

Signature:

transform(value: string): boolean

Parameters:

  • value: The URL string to check

Returns: true if the URL uses HTTPS protocol, false otherwise
